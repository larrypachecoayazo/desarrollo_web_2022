flex-grow

Capacidad de crecimiento de un elemento. Dtermina la proporción que ocupara el elemento

Si todos los artículos tienen flex-grow establecido en 1, el espacio restante en el contenedor se distribuirá por igual a todos los elementos

   flex-grow: 4; /* default 0 */

flex-shrink

Determina que cantidad de espacio disponible dentro del contenedor debe ocupar el elemento

flex-basis

Tamaño predeterminado de un elemento antes de que se distribuya el espacio restante

Combinación de flex-grow, flex-shrink and flex-basis.

   flex: none | [ <'flex-grow'> <'flex-shrink'>? || <'flex-basis'> ]

Anula la alineación por defecto (align-items) de un elemento individual
